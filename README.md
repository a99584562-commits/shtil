# Штиль

Спокойное PWA для практик дыхания и осознанности — на каждое время дня.
Никаких аккаунтов, рекламы и уведомлений. Только тихое место, чтобы вернуть
внимание из телефона в тело и в текущий момент.

## Практики

Сгруппированы по времени суток (фон приложения тоже меняется с реальным временем):

- **Утро** — когерентное дыхание 5–5, «якорь на дыхании» (медитация сосредоточения)
- **День** — бокс-дыхание 4-4-4-4, заземление 5-4-3-2-1, осознанная пауза
- **Вечер** — дыхание 4-7-8 для сна, сканирование тела (body scan / MBSR)

Каждая практика опирается на технику с исследовательской базой — короткое «почему
это работает» есть в разделе «О практиках».

«Осознанная пауза» — отдельный приём против залипания: один направленный вдох и
честный вопрос «зачем мне сейчас телефон?» перед тем, как нырнуть в ленту.

## Что внутри

- Vite + React + TypeScript + Tailwind
- `vite-plugin-pwa` — офлайн, установка на телефон, авто-обновление
- Весь звук (чаша, эмбиент, подсказки дыхания) синтезируется через Web Audio API —
  без аудиофайлов, поэтому работает офлайн и почти ничего не весит
- Прогресс (стрик, история) хранится локально в браузере, никуда не уходит

## Разработка

```bash
npm install
npm run dev      # http://localhost:5195
npm run build    # сборка в dist/
npm run preview  # локальный просмотр прод-сборки
npm run icons    # перегенерировать PWA-иконки из public/favicon.svg
```

## Установка на iPhone

1. Открой задеплоенный адрес в **Safari** (именно Safari, не Chrome).
2. Кнопка «Поделиться» → **«На экран Домой»**.
3. Запускай с иконки — приложение откроется без адресной строки, как нативное,
   и будет работать офлайн.

## Деплой

При пуше в `main` GitHub Actions собирает проект и публикует на GitHub Pages
(`.github/workflows/deploy.yml`). В настройках репозитория: **Settings → Pages →
Source → GitHub Actions**.

`base: './'` в `vite.config.ts` делает сборку переносимой — работает и на корне
домена, и на подпути вида `username.github.io/shtil/`.
